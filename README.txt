Pour ce projet # 3

J’ai utilisé juste Selenium pour faire du scrapping sur intranet de polytech. Le but final de ce projet ce de récupérer information réfèrent au différentes modules de formation à polytech Annecy. 

L’information est stockée après dans un base des données.

Les informations récupérées sont:
Modules
Responsables de chaque module
Les compétences acquis dans le module
Les pré-requis nécessaires pour entrer dans les modules.

Dans le code on peut sélectionner la formation qu’on veut récupérer l’information, dans la ligne 34  et il faut insérer un utilisateur et un mot de passe valide pour entrer au intranet de polytech dans ligne 31.
